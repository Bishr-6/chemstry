# برومبت احترافي لبناء نظام LMS المتقدم
## (Professional Prompt for LMS Development)

---

## 📝 البرومبت الرئيسي لأدوات الذكاء الاصطناعي
### (استخدمه مع: Cursor, v0.dev, ChatGPT, Claude)

```
أريد بناء نظام إدارة تعليمي متكامل (LMS) لموقع تعليمي في الكيمياء مبني بـ Next.js و Supabase.

==================================================
المتطلبات الرئيسية:
==================================================

### أولاً: لوحة تحكم المعلم (Teacher Dashboard)

#### 1. عرض النتائج والأداء
- جدول تفاعلي يعرض:
  * اسم الطالب
  * عدد الواجبات المكملة
  * متوسط النتائج
  * آخر نشاط
  * حالة الحضور

#### 2. تحليلات الاستخدام (Analytics)
- رسوم بيانية توضح:
  * عدد مرات استخدام كل أداة (حاسبة جهد الخلية، مدرب IUPAC، إلخ)
  * الأدوات الأكثر استخداماً
  * معدل المشاركة
  * الأوقات النشطة

#### 3. إدارة الواجبات (Assignment Management)
- نموذج إنشاء واجب جديد يتضمن:
  * العنوان والوصف
  * الموعد النهائي
  * نوع الواجب:
    - تسمية مركبات عضوية (IUPAC Naming)
    - حسابات جهد الخلية (Cell Voltage)
    - التحليل الكهربائي (Electrolysis)
    - معرفة المجموعات الوظيفية (Functional Groups)
    - مسائل الموازنة والحسابات
  * نقاط الواجب
  * الطلاب المستهدفين

### ثانياً: واجهة الطالب (Student Portal)

#### 1. تسجيل الدخول
- صفحة مبسطة تطلب:
  * اسم الطالب
  * رقم الفصل (اختياري)
  * اختيار المادة (الكيمياء)

#### 2. لوحة الطالب
- عرض الواجبات النشطة مع:
  * عنوان الواجب
  * الموصف والتفاصيل
  * الموعد النهائي مع عداد العد التنازلي
  * حالة التسليم
  * درجة الواجب (إن كان مقيماً)

#### 3. نظام التسليم
- إمكانية:
  * حل الواجب مباشرة عبر واجهة تفاعلية
  * تحميل ملف PDF أو صورة
  * إدخال النتائج الحسابية
  * تسجيل الملاحظات
  * تسليم والحفظ في Supabase

### ثالثاً: المختبر الافتراضي (Virtual Lab)

#### 1. قسم الكيمياء الكهربائية
- واجهة drag & drop لـ:
  * اختيار الفلزات (Cu, Zn, Al, Mg)
  * اختيار المحاليل (NaCl, CuSO₄, إلخ)
  * وضع القنطرة الملحية
  * توصيل الدائرة الكهربائية
  * قياس الجهد تلقائياً
  * تسجيل النتائج

#### 2. قسم الكيمياء العضوية
- محاكاة:
  * تفاعل الإيثاين (CaC₂ + H₂O)
  * تفاعل الإستر (حمض + كحول + H₂SO₄)
  * تفاعلات الهلجنة والهدرجة
  * تفاعلات الاحتراق

#### 3. نظام التقارير
- جدول بيانات لتسجيل:
  * الفلز المستخدم
  * المحلول المستخدم
  * الجهد المقاس
  * النسبة المئوية للخطأ
  * الملاحظات
  * التاريخ والوقت

==================================================
المواصفات التقنية:
==================================================

### Stack التقنية:
- Frontend: Next.js 16 (App Router)
- Styling: Tailwind CSS 4
- Database: Supabase (PostgreSQL)
- Authentication: Supabase Auth
- Icons: Lucide React
- Charts: Recharts أو Chart.js
- State: React Hooks + Context API

### Supabase Schema:
جداول مطلوبة:
1. teachers (id, email, name, subject)
2. students (id, name, class_id, teacher_id, created_at)
3. assignments (id, title, description, type, deadline, teacher_id, created_at)
4. submissions (id, assignment_id, student_id, content, grade, submitted_at, file_url)
5. lab_results (id, student_id, experiment_type, metal1, metal2, voltage_measured, voltage_calculated, error_percentage, notes, created_at)
6. tool_usage_logs (id, student_id, tool_name, usage_count, last_used)

### المميزات الإضافية:
✓ نظام الإشعارات عند تسليم الواجبات
✓ إمكانية المعلم لتقييم الواجبات
✓ رسائل تفاعلية للأخطاء
✓ حفظ تلقائي للمسودات
✓ تحميل الملفات والصور
✓ تصدير النتائج كـ CSV/PDF
✓ دعم اللغة العربية كاملاً

==================================================
ملاحظات مهمة:
==================================================

⚠️ استبعد تماماً أي أسئلة أو تجارب تتعلق بـ "البطاريات"
⚠️ ركز على الكيمياء الكهربائية (الخلايا الفولتية فقط)
⚠️ استخدم البيانات من "كتاب كيمياء بريدج"
⚠️ التصميم يجب أن يكون بسيطاً وواضحاً (Clean UI)
⚠️ جميع الحسابات يجب أن تكون دقيقة علمياً
⚠️ دعم الجوال الكامل (Mobile Responsive)
```
