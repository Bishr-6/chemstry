# خزانة الكيمياء الرقمية (Digital Chemistry Cabinet)

منصة تعليمية متكاملة للكيمياء الكهربائية والعضوية مع أدوات تفاعلية وقاعدة بيانات شاملة.

## المميزات الرئيسية

### 1. أدوات الكيمياء الكهربائية
- **حاسبة جهد الخلية الفولتية**: احسب جهد الخلية (E°cell) وحدد التلقائية
- **محاكي التحليل الكهربائي**: اكتشف نواتج التحليل الكهربائي للأملاح
- **دليل منع التآكل**: طرق حماية الفلزات (الجلفنة، الحماية الكاثودية، إلخ)

### 2. أدوات الهيدروكربونات
- **مدرب التسمية IUPAC**: تعلم قواعد التسمية IUPAC للهيدروكربونات
- **كاشف الأيزومرات**: استكشف الأيزومرات البنائية والفراغية والضوئية
- **المركبات الأروماتية**: البنزين ومشتقاته والخصائص

### 3. أدوات مشتقات الهيدروكربونات
- **كاشف المجموعات الوظيفية**: 11 مجموعة وظيفية مع خصائصها
- **دليل إعادة تدوير البوليمرات**: رموز التدوير (1-7) والمونومرات
- **الكيمياء في حياتنا**: تطبيقات عملية الهيدروكربونات ومشتقاتها

### 4. قاعدة البيانات الشاملة
- أكثر من 100 مادة كيميائية
- معلومات مفصلة: الصيغة، الوزن المولي، درجات الغليان والانصهار
- البحث المتقدم والتصفية حسب الفئة

### 5. لوحة تحكم المعلم
- إضافة وتعديل المواد الكيميائية
- إدارة الفئات والتجارب
- عرض الإحصائيات والتحليلات
- إدارة حسابات الطلاب

## البنية التقنية

- **Frontend**: Next.js 16 + React 19 + Tailwind CSS
- **Backend**: Supabase (PostgreSQL)
- **Hosting**: Vercel

## التثبيت والإعداد

### 1. المتطلبات
- Node.js 18+ و npm/yarn
- حساب Supabase مجاني
- حساب Vercel (اختياري للنشر)

### 2. خطوات التثبيت

```bash
# 1. انسخ المشروع
cd chemstry

# 2. ثبت المكتبات
npm install
# أو
yarn install

# 3. أنشئ ملف .env.local وأضف متغيرات البيئة
# (انسخ من .env.local الموجود وأكمل البيانات)

# 4. شغل خادم التطوير
npm run dev
# أو
yarn dev

# 5. افتح المتصفح على
http://localhost:3000
```

### 3. إعداد Supabase

#### الخطوة 1: إنشاء مشروع Supabase
1. اذهب إلى https://supabase.com
2. أنشئ حساباً جديداً
3. أنشئ مشروعاً جديداً (database)
4. احصل على `URL` و `anon_key` من الإعدادات

#### الخطوة 2: تشغيل SQL Schema

1. افتح SQL Editor في Supabase
2. انسخ محتوى `DATABASE_SCHEMA.sql`
3. ألصقه في SQL Editor واضغط "Run"

#### الخطوة 3: تحديث متغيرات البيئة

```bash
# .env.local
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

### 4. النشر على Vercel

```bash
# 1. ادفع المشروع إلى GitHub

# 2. اذهب إلى https://vercel.com

# 3. اختر "Import Project" وربط مع GitHub

# 4. أضف متغيرات البيئة في Vercel Settings

# 5. اضغط Deploy
```

## الصيغ والمعادلات المستخدمة

### 1. جهد الخلية
```
E°cell = E°cathode - E°anode
```

### 2. التلقائية
```
- E°cell > 0: تفاعل تلقائي
- E°cell < 0: تفاعل غير تلقائي
```

### 3. الصيغة العامة للألكانات
```
CnH(2n+2)
```

### 4. الصيغة العامة للألكينات
```
CnH(2n)
```

### 5. تسمية IUPAC
- اختر أطول سلسلة كربونية
- رقّم من الطرف الأقرب للمجموعة البديلة
- ضع أسماء المجموعات قبل اسم السلسلة

## الثوابت والبيانات

### الفئات الرئيسية (12 فئة)
- ألكان، ألكين، ألكاين
- عطري، هاليد ألكيل، كحول
- إيثر، أمين، كربونيل
- إستر، أميد، بوليمر
- كيمياء كهربائية

### الفلزات (10+ معادن)
مع جهود الاختزال (E°) لكل منها

### البوليمرات (7+ أنواع)
مع رموز التدوير والمونومرات

### المجموعات الوظيفية (11 مجموعة)
- الهيدروكسيل (-OH)
- الكربونيل (C=O)
- الكربوكسيل (-COOH)
- الأمينو (-NH2)
- وغيرها...

## نصائح للمعلمين

1. **بناء قاعدة البيانات**:
   - ابدأ بإضافة 20-30 مادة أساسية
   - استخدم الصيغ القياسية
   - أضف روابط للصور البنائية

2. **استخدام الأدوات**:
   - اعرض حاسبة جهد الخلية في الفصل

3. **متابعة الطلاب**:

## استكشاف الأخطاء

### المشكلة: البيانات لا تظهر من Supabase
**الحل**:
1. تحقق من متغيرات البيئة
2. تأكد من تشغيل SQL Schema
3. تحقق من صلاحيات الجداول

### المشكلة: الصفحة لا تحمل
**الحل**:
1. افتح Developer Console (F12)
2. تحقق من الأخطاء
3. أعد تشغيل خادم التطوير

### المشكلة: الأدوات لا تعمل بشكل صحيح
**الحل**:
1. تحقق من دوال lib/calculations.js
2. تأكد من مطابقة الثوابت
3. اختبر على متصفح مختلف
